# 🚀 Laravel Professional Development Workflow  
### Docker + Git + CI/CD (GitHub Actions)

This project demonstrates a **modern Laravel development workflow** used by professional teams.  
It integrates **Docker for environment consistency**, **Git for version control**, and **GitHub Actions for CI/CD automation**.

The goal of this repository is to showcase a **production-ready Laravel setup** with automated testing, containerized infrastructure, and continuous integration.

---

# 📌 Tech Stack

- **Laravel 12**
- **Docker & Docker Compose**
- **MySQL**
- **Redis**
- **Nginx**
- **Node.js (Vite)**
- **Git & GitHub**
- **GitHub Actions (CI/CD)**
